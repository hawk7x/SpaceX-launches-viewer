A React web application that displays **SpaceX launch data** using the public SpaceX REST API.

The project demonstrates practical React skills, API integration, state management with hooks, and a clean user interface.

--- 

## Features

- Fetches SpaceX launches from a public API
- Search launches by mission name
- Launch details displayed in a modal window
- Webcast link for each launch (if available)
- Loading and error handling states
- Responsive dark-themed UI

---

## Tech Stack

- React (Hooks, Functional Components)
- JavaScript (ES6+)
- CSS
- SpaceX REST API
